Base mapping providers

PATHFINDER uses free providers of base raster type cartography. List shows below.

CartoDB. Software as a Service (SaaS) cloud computing platform providing GIS and web mapping tools to view in a web browser.
Esri. Founded by Jack Dangermond as Environmental Systems Research Institute (Esri) in 1969 as a land-use consulting firm, is an international supplier of geographic information system (GIS) software, web GIS and geodatabase management applications. The company is headquartered in Redlands, California..
Hydda, OpenStreetMap (OSM), OpenTopoMap, derivations from the same project (OSM), create and manage freely accessible editable maps. The size of the database (planet.osm) was in July 2017, above 800 gigabytes.
Stamen. Using OSM as functional and cartographic base, Stamen gives format, sometimes in a visually very suggestive, attractive and useful way, public access maps.
Wikimedia. Repository of media files putting not-only-textual content in public domain, including maps, publicly available in numerous languages.
