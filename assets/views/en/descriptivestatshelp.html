Legend of the table. Descriptive statistics

In each data operation carried on by PATHFINDER, a Stats button opens a screen showing descriptive statistics info, being able to be used as help to better understanding of distributive conditions of the analyzed sample. Meaning for abbreviations used as table header are shown below, but feel free to refer the documentation to learn more.

FLD Referenced field.
TYP Field content type: numeric, string, date,…
FRQ Data frequency related indicators.
  • VAL: total values (number of total rows retrieved).
  • DST: distinct values (number of rows containing non-repeated values).
  • MIS: missing values (number of empty or not-value-containing rows).
RNG Indicators relating to the range of values covered by the data.
  • MIN: minimum value.
  • DST: maximum value.
  • (N): otal range between minimum and maximum value.
AVG Indicators referring to values showing a central trend in the dataset.
  • MEA: arithmetic mean.
  • MED: median.
  • STD: standard deviation.
QRT Indicators referring to the extremes determining the presence of significant values.
  • Q1: first quartile limit.
  • Q3: third quartile limit.
  • MSK: mode asymmetry.
Entropy Referent of interest when evaluating the importance and significance of the data contained in a field. Query the documentation about.
D y J Mutual distance and Jaccard index. Measurement of degree of affinity or similitude between frequencies for a single field. Query the documentation about.
