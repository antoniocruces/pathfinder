13. COOKIES POLICY

PATHFINDER uses its own cookies for technical reasons. In no case does it use third-party cookies, except those strictly necessary for access measurement (Google Analytics ).

According to the Directive 2009 / 136 / EC on ePrivacy and the Regulation (EC) 45 / 2001 on data protection, as specified in this regard by the Spanish Data Protection Agency in your Guide on the use of cookies , the user of PATHFINDER may accept or decline the use of this mechanism.

A cookie is a small piece of data that a web page inserts, through the client user's browser, that it saves on its computer or mobile device. The cookie allows the website to "remember" your actions or preferences over time. Most browsers support cookies, but users You can configure your browsers to reject them and you can delete them whenever you want.

Websites mainly use cookies to:

  • Identify the users.
  • Remember user preferences.
  • Help users complete tasks without having to re-enter information when navigating from one page to another or visiting the site later.

Cookies can also be used to advertise behavioral objectives online and to display relevant ads for something the user searched in the past.

How are they used?

The web server that supplies the web page can store a cookie on the user's computer or mobile device. An external web server that manages files included or referenced on the website can also store cookies. All these cookies are called "HTTP header". Another way to store cookies is through the JavaScript code contained or referenced on a page. Each time the user requests a new page, the web server can receive the values of the previously established cookies and return the page with content related to these values. Similarly, the JavaScript code is able to read a cookie belonging to your domain and perform an action accordingly.

What are the different types of cookies?

A cookie can be classified by its useful life and the domain to which it belongs. For duration, a cookie can be:

  • Session cookie, deleted when the user closes the browser.
  • Persistent cookie that remains on the user's device / device for a predefined period of time.

As for the domain to which it belongs, there are:

  • First cookies that are set by the web server of the visited page and share the same domain.
  • Third-party cookies stored in a domain other than the domain of the visited page. This may occur when the web page refers to a file, such as JavaScript, that is outside your domain.

What type of cookies does PATHFINDER use?

  • Technical cookies: These are those that allow the user to navigate through a web page, platform or application and use the different options or services that exist there, such as, for example, control traffic and data communication, identify the session, access restricted access parts, remember the elements that make up an order, make the purchase process of an order, make the request for registration or participation in an event, use security elements during navigation, store content for dissemination of videos or sound or share content through social networks.
  • Personalization cookies: These are those that allow the user to access PATHFINDER with some predefined general characteristics based on a series of criteria in the user's terminal, such as the language, the type of browser through which the service is accessed, the regional configuration from where you access the service, etc.
  • Analysis cookies: Those that are well treated by PATHFINDER or by third parties, allow quantifying the number of users and thus perform the measurement and statistical analysis of the use made by users of the service offered. To do this, we analyze your navigation in PATHFINDER in order to improve the offer of services you provide.
