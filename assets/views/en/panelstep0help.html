1. ACTUALIZACIÓN

PATHFINDER utiliza para trabajar archivos procedentes del servidor de datos ExpoFinder. Dichos archivos se descargan desde Internet. Tras un periodo de duración variable (por lo general entre 30 y 60 segundos), PF habrá descargado un paquete de información en formato JSON codificado e indicará un número PIN, que se deberá conservar para poder acceder a los datos en un futuro.

El sistema indica el estado del servidor de datos (en color verde si está activo) y el tiempo calculado de conexión en milisegundos (ms).

Para obtener los datos debe disponer de un juego de credenciales válido (es decir, un nombre de usuario y una contraseña) para acceder a ExpoFinder. Si no dispone de uno, solicítelo aquí.
