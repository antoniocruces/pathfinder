PATHFINDER needs to load data as a JSON file with the appropriate format. Go to the data page if you have not a file.

  • If you do not have a JSON file with the correct format or you want to update data from ExpoFinder server, perform a data update process.
  • If you have a JSON file with the correct format, proceed to select that file. PATHFINDER will load your information into the database and then you will be able to run the app.
