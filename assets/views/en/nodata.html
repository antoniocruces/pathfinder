PATHFINDER necesita cargar los datos presentes en un archivo JSON con el formato adecuado. Si no ha realizado dicha carga, diríjase a la la página de datos.

  • Si no dispone de un archivo JSON con el formato adecuado o desea actualizar los datos extraídos del servidor ExpoFinder, realice una actualización de datos.
  • Si dispone de un archivo JSON con el formato adecuado, proceda a seleccionar dicho archivo. PATHFINDER cargará su información en la base de datos y podrá operar con ellos.
