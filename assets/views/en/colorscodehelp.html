Color codes

PATHFINDER uses a color coded system to identify different types of elements. PATHFINDER utiliza un sistema codificado de colores para identificar diferentes tipos de elementos. The primary record types are identified according to the table below.

P Actor
B Catalog
C Company
E Entity
X Exhibition
A Artwork

In PATHFINDER, each type of link or button has a color code shown as a thick line on the left, indicating the type of activity it will perform. This code corresponds to the following table.

  Generation of files and download to disk. PATHFINDER creates a file containing the requested information and downloads it to a disk on the client’s computer. For proper operation, of course, this computer must have sufficient free space.
  Downloading files or information from the server. You need the client’s computer to be connected to the Internet. In the case of downloading files, this computer must have sufficient free space.
  Presentation of information prepared or calculated by PATHFINDER. Can be displayed in a pop-up window (sometimes in full screen). Usually involves a variable processing time.
  File upload from the client’s computer disk. Normally the files consumed by PATHFINDER are usually in JSON format.
  Help information. This is usually in the form of a pop-up window.
  Printing or downloadig image or Adobe™ Acrobat® (PDF) file.
  [NOT YET SET].
