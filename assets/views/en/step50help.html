1. INCLUSIONS & RESTRICTIONS.

They affect the records that the filter will select once the filtering conditions have been applied.

  • Include. Add to the list of primary records selected by the filter conditions those directly related to them compliant with the checked types. For example, if conditions only affect exhibitions but actors are included, those related to each filtered exhibition will be added.
  • Restrictions. Limit the records included only to those ones whose relational field is checked. Following the previous example, it is possible to select only those actors acting as exhibition curators.

2. FILTER PANEL.

Each condition is represented by a row, with the following features.

  • Modifier. Select the facet to be used to validate the condition. For example, "year" in the case of a date.
  • Field on which the search will be carried out.
  • Operator. The usual ones to compare numerical or textual values, considering or not (in the case of contains and does not contain operators) uppercase and diacritical, plus the between operator, designed to deal with two numerical values in sequence ("between 2010 and 2015" will select the records whose year, if that facet was applied, falls within that range), although it also works with textual values.
  • Value. It is possible to get a list of matching possible values by clicking on icon.
  • Search (red if no result) and remove condition buttons.

There is no limit on the number of applicable conditions.

3. OPERATIONS.

Button panel at the bottom of the filter window allowing:

  • Save or load "collections", which are frequently used filter definitions, that the user can store as files on his local disk.
  • Calculate. It means, to perform single calc to each condition.

    IMPORTANT: PF will not perform the filtering operation if all present conditions have not been calculated.

  • Filter. Evaluation of the conditions and generation of the filtered subset. When done, an additional button is displayed to present a graph of the conditions and their links.
  • Clear. Set up the filter by deleting the subset.
