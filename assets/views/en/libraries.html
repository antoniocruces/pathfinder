Code libraries

According to Wikipedia, a library —in the computer sense of the word— is "a set of functional implementations, encoded in a programming language, which offers a well-defined interface to the functionality that is invoked".

PATHFINDER uses the following libraries to optimize system performance, depending on the page or section you are working with at all times:

  • Datalib v1.8.0 [dl]. . It is a JavaScript data utility library providing facilities for loading data, type inference, common statistics and string templates. Licenses: BSD 3-Clause "New" or "Revised"
  • Leaflet v1.3.1 [lj]. . It is a leading open source JavaScript library for optimized interactive maps. Licenses: BSD 2-Clause
  • Lodash v4.17.10 [lo]. . It is a modern library of JavaScript utilities that offers modularity, performance and extras. Licencses: MIT
  • ECharts v4.1.0 rc2 [ec]. . It is a powerful graphics library and free code visualization that offers an easy way to add intuitive, interactive and highly customizable graphics to any application. Licenses: Apache 2.0
