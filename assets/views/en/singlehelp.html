Screen items

This screen shows a single primary record. Info is grouped according to their semantic value in the general context of the application and particular of the register itself.

Top to bottom, the screen areas are as follows:

  1. Record title ("Untitled" if blank).
  2. Single screen menu
    1. Record. Returns to basic record screen.
    2. Analysis. Shows analytical to apply to the record data.
      1. Timeline. Line chart showing outstanding chronological marks of the record. It is possible to characterize it using facets.
      2. Geoline. Line chart showing outstanding geographical marks of the record. It is possible to characterize it using facets.
      3. Network. Graph showing relationships of the record. It is possible to use up to four graph layouts (simple, circular, force or sankey) and to characterize it using facets.
      4. Taxogram. Line or radar chart showing outstanding taxonomies marks of the record. It is possible to characterize it using facets.
      5. Stats. Table showing absolute and relative frequences of metainfo of the record field-by-field basis.
      6. Maps. It is possible to perform same set of operations present at general maps menu, but restricted to selected record only.
    3. Legend. Shows ID number for the selected primary record and colors code used along whole app.
  3. Groups for each type of metadadata: Taxonomies, Metainfo and Relationships. In this case, each bunch of relations is sorted by relation kind and related primary record, by the way using aforementioned colors.

On the right side of the screen header there is an icon . It allows to download the plain text content of the whole screen.
