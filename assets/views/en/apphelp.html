FAQ

What essential steps should I take to start using PATHFINDER?
They are very few, and quite simple to follow:
  1. Make sure that you own a name of a valid ID in ExpoFinder. You can get one making contact with de development team.
  2. Download the file from the data server and save it in a place known in your computer. Remember that is VERY IMPORTANT to remember the provided PIN: without it you would not decode the JSON file.
  3. Open the saved file. PATHFINDER will ask you for the provided PIN.
  4. Consult the documentation in order to know which are the operations you can make with PATHFINDER and how you can make them.
Is it necessary an active Internet connection in order to use PATHFINDER?
Only to download or renew the JSON file used to prepare the data base, which must have been generated by ExpoFinder.
Is it essential to have a very powerful equipment at my disposal in order to work with PATHFINDER?
Not at all. In fact, it is not the usual “program” which is necessary to “install” in the client’s computer, but it uses the browser of the computer and it works by scripts. We have tried to keep to the paradigm of Poor Man Computing, praised by Alexander Shchepetkin, I G.P.P UCLA, among others, on his conference in Venice in October 2004. It is about doing the most as possible with the simplest tools and at the reach of anyone. In fact, it is not necessary to acquire additional hard or soft to use it.
PATHFINDER has been tested in hard conditions, with little RAM available, with diverse operative systems, (Windows, macOS, Linux) and with a broad group of browsers (Chrome, Firefox, Opera, Safari). Visit the requirements page to find out. The only indispensable thing is to have an updated version of the browser you will use. PATHFINDER uses exceptional technology (ECMAScript 5, 6 and a few elements of the 7) which leverage the most developed programming characteristics. An obsolete browser will not work well, it will make errors and it will not allow to obtain the proper efficiency.
Can I use the data belonging from ExpoFinder for my research and publish them?
PATHFINDER legal terms stablish so. Its main goal is to be a helping tool for the investigation in the Digital History of Art and Digital Humanities environment. However, you will need to have a permission at your disposal in order to do that, mainly because you will not have access to updated data if you do not own a set of credentials from ExpoFinder. In the legal page you can locate the information needed.
What can I do if I lose or forget the access PIN to the JSON file?
PATHFINDER uses data from ExpoFinder in a file encoded by a symmetric algorithm, simple but efficient, before being stored in the disc of the client’s computer. It would not decode it if you do not own that password. In that case, you will need to make a new download.
What can I do if an error appears while I try to make an operation?
In the first place, take note of the message shown on the screen. Usually the information about the error is simple, but it is possible to make it more detailed. For that, go to the settings page and in the section Settings → Errors control → Detailed error messages mark yes. Once you obtain the “trajectory” of the error, take note and send the copy to the PATHFINDER development team using for that the contact page.
Once I open the download JSON file from ExpoFinder it gives me error saying that the registration is not correct. What can have caused this?
Mainly four things:
  1. Sometimes throughout the data download from the server, there is a chance that an unnoticed network error causes an imbalance of some type. In that case, the file will save with that error and it could not be used. You will need to download it again.
  2. If an incorrect ExpoFinder credentials set is specified in order to identify yourself in the data server, PATHFINDER “silently fails”, so it will not alert any intruder in and it will simply download a file with incorrect content. If you try to access it and charge it in the database an error is made.
  3. The PIN that you specified when you open the file is not correct. Doing that, PATHFINDER obtains a wrong decoding and the values that the system search for are not correct. Try the operation again and make sure that the PIN is the appropriate one.
  4. You have tried to open a file which has not been downloaded in the ExpoFinder server and obviously it does not contain the relevant information. Make the download again.
Does PATHFINDER take long to charge the database?
During the charging process the app makes numerous operations, among them “cleaning” the erroneous searches and calculate the age from those searches. In general terms, even these numbers depend on microprocessor of the client’s computer, PATHFINDER processes between 37000 and 45000 searches per second, so 500000 searches are processed in a lapse that oscillates between 11 and 13,5 seconds. Filtering operations, pagination and search are below 1 second for the data volume although it depends on the complexity of the filtering conditions.

How can I…

…narrow down the limits of the research?
The best way is using a filter. For that, you will need to access to the data page, make sure that you have a valid JSON file, charge it in the database and open it in list. The headline of the paged table shows two symbols: [+] to add conditions to the filter and [-] to remove all the conditions to the filter. If you do the last one PATHFINDER will return to its initial state, in which every search will be considered and the work field is not delimited by the creation of a subset of data or dataset.

Remember that the conditions are logical and they apply making operations of intersection, union or difference of sets (“datasets”). Consult the specific help section in the filter window to obtain specific instructions of how to make them.
