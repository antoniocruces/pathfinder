Export

PATHFINDER can generate files containig the data it handles and they can be read by other applications. The process of export is very simple. Select the format you want and click the adequate button. Note that PATHFINDER will only export records being filtered.

TSV format.

One record per line. Fields separated by a tab (\t) character.

See: IANA Media Types

TSV

CSV format.

One record per line. Fields separated by a comma (,) character.

See: IETF RFC 4180

CSV

XML fomat.

Extensible Markup Language.

See: IETF RFC 7303
