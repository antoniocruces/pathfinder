PATHFINDER can generate files with the database content, ready to be read by other applications. Export process is very simple. Select the format you want and click on the appropriate button.

Note that PATHFINDER will only export the records that have been filtered.

TSV Format

Export in TSV format

One record per line. Fields separated by a tab character.

References: IANA Media Types

CSV Format

Export in CSV format

One record per line. Fields separated by a comma (,) character.

References: IETF RFC 4180

XML Format

Export in XML format

Extensible Markup Language. Predefinited tags.

References: IETF RFC 7303
