Utilice estas herramientas de estadística descriptiva para realizar cálculos sobre el conjunto total de datos a algún subconjunto. Podrá obtener medidas de dispersión, frecuencia, variación y otros valores que permitirán estudiar la representatividad y comparar la muestra y el universo estudiado.

Utilice estas herramientas para realizar cálculos de estadística inferencial sobre el conjunto de datos gestionado por PATHFINDER. Podrá realizar, entre otras operaciones…

  • muestreos que permita obtener conclusiones estadísticamente válidas y significativas
  • identificación de variables estadísticas que permitan estimar valores poblacionales a partir de muestras de mucho menor tamaño
  • contraste de hipótesis para decidir si dos muestras son estadísticamente diferentes o si un determinado procedimiento tiene un efecto estadístico significativo
