Bibliotecas de código

Según Wikipedia, una biblioteca —en el sentido informático de la palabra— es "un conjunto de implementaciones funcionales, codificadas en un lenguaje de programación, que ofrece una interfaz bien definida para la funcionalidad que se invoca".

PATHFINDER utiliza las siguientes bibliotecas para optimizar el rendimiento del sistema, dependiendo de la página o sección con la que se esté trabajando en cada momento:

  • Datalib v1.8.0 [dl]. . Es una biblioteca de utilidad de datos de JavaScript que proporciona instalaciones para la carga de datos, inferencia de tipos, estadísticas comunes y plantillas de cadenas. Licencia(s): BSD 3-Clause "New" or "Revised"
  • Leaflet v1.3.1 [lj]. . Es una biblioteca de JavaScript de código abierto líder para mapas interactivos optimizados. Licencia(s): BSD 2-Clause
  • Lodash v4.17.10 [lo]. . Es una biblioteca moderna de utilidades de JavaScript que ofrece modularidad, rendimiento y extras. Licencia(s): MIT
  • ECharts v4.1.0 rc2 [ec]. . Es una potente biblioteca de gráficos y visualización de código libre que ofrece una forma fácil de agregar gráficos intuitivos, interactivos y altamente personalizables a cualquier aplicación. Licencia(s): Apache 2.0
